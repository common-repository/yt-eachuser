== Description ==

The list of video is displayed specifying user of Youtube.
The display form and the display sizes such as thumbnails and players can be selected.
* * *

**shortcode**
[print_yt_EachUser]
* ytname : youtube user name
* type : player or thumbnail
* width : width px (integer)
* height : height px (integer)
* num : view count (integer)
* api_key : Google API key (string)


example:
[print_yt_EachUser ytname="YouTube" type="player" width="425" height="355" num="3"]
